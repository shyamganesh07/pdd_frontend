Google Sign-In Configuration

To retrieve your active Chrome Google ID, you need a free OAuth Client ID from Google Cloud.

  1. Go to the Google Cloud Console.
  2. Create a project and configure the OAuth Consent Screen (External/Testing).
  3. Go to Credentials → Create Credentials → OAuth client ID.
  4. Choose Web application.
  5. Add Authorized JavaScript origins:
    http://localhost:5173
  6. Add Authorized redirect URIs:
    http://localhost:5173/google-sign-in.html
  7. Copy your Client ID and paste it below.
Google OAuth Client ID

Tip: You can also specify this globally by adding GOOGLE_CLIENT_ID in your backend .env file.
